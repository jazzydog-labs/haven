# Haven Documentation

Welcome to Haven - a self-contained microservice exposing REST and GraphQL APIs over PostgreSQL.

## What is Haven?

Haven is a modern Python microservice built with:

- **FastAPI** for high-performance REST APIs
- **Strawberry GraphQL** for type-safe GraphQL
- **SQLAlchemy 2.0** with async support
- **Clean Architecture** principles
- **Comprehensive testing** and quality gates
