# Haven Documentation

Welcome to Haven - a self-contained microservice exposing REST and GraphQL APIs over PostgreSQL.

## What is Haven?

Haven is a modern Python microservice built with:

- **FastAPI** for high-performance REST APIs
- **Strawberry GraphQL** for type-safe GraphQL
- **SQLAlchemy 2.0** with async support
- **Clean Architecture** principles
- **Comprehensive testing** and quality gates

## Key Features

### 🚀 Modern Tech Stack
- Python 3.12+ with full type hints
- Async/await throughout
- Pydantic v2 for validation
- Hydra for configuration

### 🏗️ Clean Architecture
- Clear separation of concerns
- Domain-driven design
- Repository pattern with Unit of Work
- Dependency injection

### 📡 Dual API Support
- RESTful API with OpenAPI/Swagger
- GraphQL with schema introspection
- Consistent data models across both

### 🧪 Quality First
- 70%+ test coverage requirement
- Strict type checking with Pyright
- Automated linting with Ruff
- Pre-commit hooks

### 🐳 Production Ready
- Docker support with minimal images
- Health checks and monitoring
- Structured logging
- Database migrations with Alembic
